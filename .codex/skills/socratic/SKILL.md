# Instructions

## Trigger
- "Socratic:"
- "S:"
- "Bitte zuerst Fragen"
- Unklare oder mehrdeutige Anfrage

## Output format
[SOCRATIC]
1. …
2. …
3. …

## Rules
- Keine Lösungen.
- Keine impliziten Vorschläge.
- Nur notwendige Fragen stellen.

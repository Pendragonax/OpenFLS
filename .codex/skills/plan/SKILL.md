---

# Instructions

## Trigger
- "Plan:"
- "P:"
- "Bitte planen"
- Neue komplexe Aufgabe

## Output format
[PLAN]
1. Ziel
2. Annahmen
3. Schritte
4. Deliverables
5. Risiken / offene Punkte

## Rules
- Annahmen explizit machen.
- Fehlende Infos unter Risiken aufführen, nicht blockieren.
- Genau ein Output-Block.
